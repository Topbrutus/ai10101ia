# Code de conduite

## Engagement

Ce dépôt vise un travail rigoureux, traçable et respectueux.

## Comportements attendus

- communiquer de façon claire, précise et respectueuse
- documenter les décisions difficiles
- critiquer les choix, pas les personnes
- corriger sans dissimuler les impacts ni les preuves
- respecter les validations imposées par le dépôt

## Comportements interdits

- contournement volontaire des règles du dépôt
- suppression opaque d’éléments imposés
- dissimulation de risques ou de preuves
- sabotage, intimidation, harcèlement ou mépris

## Arbitrage

Les arbitrages passent par les mécanismes décrits dans `docs/GOVERNANCE.md`.
